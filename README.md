# GenerateLyricCard
### Purpose
- for English Learners
- This program is inspired by "Parrot's Law" from "Eigomimi"(in Japanese: 「英語耳」, in English: "English Ear").
- This program generate lyric card with phonetic symbols from text of lyric to PDF file.
### Specification
- what you need
    - Python3
    - OpenCV
    - Necassary libraries
        - cv2
        - numpy
        - requests
        - BeautifulSoup
